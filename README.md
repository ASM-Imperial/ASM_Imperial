# Node.js + Vite Project

This project uses Node.js for the server-side and Vite for the frontend build tool. Vite is a modern, fast, and efficient build tool that leverages native ES modules for development.


---

## Installation

To get started, clone the repository and install the required dependencies.

### 1. Clone the Repository

```bash
git clone https://github.com/mndNepal/ModernNatureDesginNepal.git
cd ModernNatureDesginNepal
```

### 2. Install Dependencies

```bash
npm install
```
### 3. Development

To start the development server with Vite and run the project locally, use the following command:
```bash
npm run dev
```
